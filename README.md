# Ekop Designz Hub

## Project Overview
Ekop Designz Hub is a collaborative platform that brings together designers, developers, and creative thinkers. It allows users to share their design projects, collaborate on ideas, and find inspiration.

## Features
- User authentication and profiles
- Project sharing and collaboration tools
- Community feedback and reviews
- Design inspiration and resources
- API access for developers

## Tech Stack
- **Frontend:** React, Redux, CSS, HTML
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Authentication:** JWT & OAuth support
